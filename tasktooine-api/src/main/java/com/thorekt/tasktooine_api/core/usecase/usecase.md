# usecase

Contient les cas d'utilisations métier du projet ainsi que la définition des interfaces requises par ces cas d'utilisation

Un cas d'utilisation est défini tel que :
- Class
- L'injection des dépendance requise au cas d'utilisation
- Une méthode "execute" qui éxécute la logique métier
- Hérite du l'interface IUseCase
