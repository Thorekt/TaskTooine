# gateway

Contient les gateways du projet.

Une gateway est définie tel que :
- Class
- Implémente les interfaces requises par les cas d'utilisation
- Gère l'accès aux ressources externes au domaine métier
- Contient la logique technique de communication avec la base de données, une API externe, un système de fichiers, etc.
- Ne contient pas de logique métier
